# Wheater Forecast
Wheater Forecast is mini app for wheather forecast. it has 3 tabs, in first tab user can see current location wheather and wheater hourly. You can change your location by clicking in  top right corner button and search from cities list. In the left top corner is current location button.
in the third tab is forecast for 7 days for current location and it is available in ofline mode.

# Development

App is written in swift with MVVM architecture.For wheather forecast used http://openweathermap.org/ open Api. Api request is done by Alamofire framework. Cities search is done localy by city.json file
